# Abeselom's Python Adventure 🎮

A 4-week Python course for a beginner who loves games. No programming experience needed!

## How to set up (one time, with a parent)

1. Go to [python.org/downloads](https://www.python.org/downloads/) and download Python.
2. When installing on Windows, **check the box "Add Python to PATH"**.
3. Install a free code editor: [Thonny](https://thonny.org/) is perfect for beginners (it's made for learning Python).

## How to use this course

- Each week has its own folder with a **LESSON.md** file (read this first) and **practice `.py` files**.
- Open the `.py` files in Thonny and press the green **Run** button.
- Do the lessons in order. Spend about **30-45 minutes, 3 days a week**.
- The `solutions` folder in each week has answers — try first before peeking!

## The Plan

| Week | Topic | Game you'll build |
|------|-------|-------------------|
| 1 | Talking to the computer (print, variables, input) | Mad Libs story game |
| 2 | Making decisions (numbers, if/else) | Quiz game |
| 3 | Repeating things (loops, random) | Number Guessing game |
| 4 | Putting it all together | Rock, Paper, Scissors |

## Rules for success

1. **Type the code yourself** — don't copy-paste. Your fingers learn too.
2. **Errors are clues, not failures.** Every programmer sees errors all day. Read the message — it usually tells you which line to check.
3. **Change things and see what happens.** That's how gamers explore, and it works for code too.

Have fun! 🐍
