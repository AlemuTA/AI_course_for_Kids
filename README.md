# AI_course_for_Kids
From Curious to Capable: An AI Course for Young Minds
A complete, self-paced curriculum that takes a young learner from total beginner to confident AI builder over roughly 3–6 months. Across 8 modules, kids train real machine-learning models in the browser, explore how neural networks learn, write their first Python, build AI-powered apps, and learn to use tools like chatbots wisely and safely. Every activity uses free tools, and each module includes notes for parents facilitating the journey.
Topics covered: AI fundamentals · machine learning · data & bias · neural networks · coding (Scratch & Python) · generative AI & LLMs · ethics & critical thinking · capstone projects.
